# 3D_Squared-OC-themes
Themes for use with OpenCore 0.7.0 and Higher and OpenCore Legacy Patcher (OCLP)


## First Use ➤  Change `config.plist` in `EFI/OC`

### Set PickerVariant (3D_Squared_Hero theme for example)

- Misc —> Boot —> PickerVariant: Alphascorp\3D_Squared_Hero

### Set PickerAtributes

- Misc —> Boot —> PickerAtributes: 145


## Replace the unziped `Resources` folder in `EFI/OC`
