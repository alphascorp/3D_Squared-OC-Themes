# 3D_Squared-OC-themes
#### Themes for use with OpenCore 0.7.0 and Higher and OpenCore Legacy Patcher (OCLP)

---

## :warning: First Use ➤  Change `config.plist` in `EFI/OC`

### Set PickerVariant (3D_Squared_Hero theme for example)

- Misc —> Boot —> PickerVariant: Alphascorp\3D_Squared_Hero

### Set PickerAttributes

- Misc —> Boot —> PickerAttributes: 145

Like this: :arrow_heading_down:
```
<key>Misc</key>
<dict>
  
  …
  
  <key>Boot</key>
  <dict>
    
    …
    
    <key>PickerAttributes</key>
    <integer>145</integer>
    
    …
    
    <key>PickerVariant</key>
    <string>Alphascorp\3D_Squared_Hero</string>
    
    …
          
  </dict>
```

## Replace the unziped `Resources` folder in `EFI/OC`
